SVN 的使用

Visual SVN 的下载使用

1— 安装完成打开 Visual SVN Server 即可看到界面。
2— 然后就可以新建仓库，选择 FSFS 的。
3— 建议选择 empty 选项，然后所有人可以访问，当然也可以选择其他的。
4— 创建用户用于访问仓库，也可以创建组对用户分类。
5— Copy URL to ClipBoard 用于在浏览器访问。

Tortoise SVN 的下载使用

右键 TortoiseSvn Repo-browser ，把 URL 地址给他即可。然后输入账号密码，以后登录自动存储。要清除这些保存的账号密码：右键 TortoiseSvn settings saved data clear ，

第一次从 svn 服务器下载代码称为 checkout， 右键 SVN Checkout 即可

add 表示添加到本地仓库，commit 表示提交到服务器。显示历史版本：show log

解决冲突：同时对一个文件操作，一个提交，一个不提交。

企业管理系统环境搭建

1— 新建工程
2— 新建模块
1— 新建ssm-dao
2— 新建ssm-service
3— 新建ssm-domain
4— 新建ssm-utils
5— 新建ssm-web ，此时要注意，创建这个模块要选择骨架：maven-archetype-webapp
6— 在 ssm-web/src/main 下新建 java 和 resources 文件夹，然后标记作用
7— 然后再resources下面加入3个文件：
1— applicationContext.xml Spring 的配置文件
2— db.properties 数据库配置文件
3— spring-mvc.xml Spring MVC 配置文件

解决 Oracle 问题：
1— java.sql.SQLException: No suitable driver
2— java.lang.ClassNotFoundException: oracle.jdbc.driver.OracleDriver

出现这两个问题的原因是没有添加依赖，并且版本不能用数字版本如 11.2.0.1.0 ，应该用 release 版本，这样才可以运行项目。
